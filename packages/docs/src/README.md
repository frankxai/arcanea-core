## 🚀 Quick Start

Manifest your first multiverse in minutes:

```bash
# Install the ARCANEA CLI
npm install -g @arcanea/cli

# Create your first realm
arcanea create my-business-realm --template business-platform

# Navigate to your realm
cd my-business-realm

# Install dependencies
npm install

# Activate the Six Forces
npm run dev
```

Visit `http://localhost:3000` to see your multiverse manifestation platform!

## 🏗️ Realm Creation Example

Here's how easy it is to manifest a complete business realm:

```typescript
import { RealmBuilder, MultiverseEngine } from '@arcanea/six-forces'

// Create a comprehensive business platform
const businessRealm = await new RealmBuilder()
  .createRealm()
  .withVision("Revolutionary e-learning platform that makes complex topics accessible through AI-powered personalized learning paths")
  .withTemplate('education-platform')
  .withForces([
    'flame',        // Business strategy and monetization
    'form',         // User interface and visual design
    'lore',         // Educational content and storytelling
    'resonance',    // Audio learning and accessibility
    'synthesis',    // System architecture and integrations
    'manifestation' // Physical learning materials and events
  ])
  .withTarget('hybrid') // Digital platform + physical components
  .withTimeline('comprehensive')
  .manifest()

// Result includes:
// - Complete business plan and strategy
// - Full UI/UX designs and prototypes
// - Course content and learning materials
// - Audio branding and accessibility features
// - Technical architecture and development roadmap
// - Implementation plan including physical components
```
